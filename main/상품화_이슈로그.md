## #001 | 2026-03-29 | 서브에이전트 미연결

**문제:** 부서별 에이전트를 만들었지만 openclaw.json에서 main의 서브에이전트로 등록하지 않음. 오케스트레이션(session_send) 불가.
**원인:** 에이전트 생성과 서브에이전트 연결이 별도 작업인데, 연결 단계를 빠뜨림.
**해결:** openclaw.json의 main 에이전트에 subagents.list 추가.
**일반화:** ✅ 모든 고객이 겪을 문제. 설치 스크립트에 자동 연결 포함 필요.

## #002 | 2026-03-29 | Claude Code PATH 문제

**문제:** AGENTS.md에 `claude --print`로 호출 지시했지만, OpenClaw exec 환경에서 PATH 미등록으로 command not found.
**원인:** npm global 설치 경로가 쉘 PATH에 포함 안 됨.
**해결:** 전체 경로 `/Users/mac/.npm-global/bin/claude`로 변경.
**일반화:** ✅ 설치 환경마다 경로 다름. 설치 시 자동 감지 필요.

## #003 | 2026-03-29 | GitHub repo 중복 생성

**문제:** 기존 ai-employee-memory에 이미 조직 구조가 있는데, 별도 repo(puttist-ai-org)를 만들어 기존 텔레그램 알림/동기화 연결 끊어짐.
**원인:** 기존 인프라 연결 확인 없이 새 repo 생성.
**해결:** 기존 repo에 통합, 별도 repo 삭제.
**일반화:** ✅ 고객 가이드에 "기존 연결 확인 후 작업" 체크리스트 필요.
