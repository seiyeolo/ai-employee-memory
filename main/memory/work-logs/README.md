# 작업 일지 자동 기록 시스템

## 사용 방법

### 기록하기 (편하게 말하기)

**텔레그램에서 이렇게만 말씀하세요:**

```
세미, 오늘 생산 50개, 발송 30건

세미, 생산 20개 했고 재고 15개 남음

세미, 택배 40건 보냈어

세미, 오늘 작업 - 아침 생산 30개, 오후 발송 25건, 재고 정리
```

**어떤 표현도 가능:**
- "생산 50개"
- "50개 만들었어"
- "오늘 50개 생산"
- "생산량: 50"

### 확인하기

```
세미, 오늘 뭐했지?
세미, 이번 주 생산량은?
세미, 이달 발송 건수 알려줘
세미, 1월 작업일지 써줘
세미, 분기 리포트 만들어줘
```

## 제품 및 작업 분류

### 제품 분류
- **퍼티스트 골프**: 고급형, 일반형
- **퍼티스트 파크**: 고급형, 일반형

### 작업 분류
- **부품생산**: 센서, 레이저 등 부품 제작
- **생산조립**: 본체 조립
- **포장**: 퍼티스트 골프, 퍼티스트 파크 완제품 포장

## 자동으로 기록되는 항목

### 부품생산
- 센서, 레이저 등 부품 수량
- 담당자
- 작업 시간

### 생산조립 (본체)
- 제품 종류 (골프/파크, 고급형/일반형)
- 수량
- 담당자
- 작업 시간

### 포장
- 제품 종류 (퍼티스트 골프/파크)
- 수량
- 담당자

### 발송
- 발송 건수
- 택배사 (선택)
- 배송 지역 (선택)

### 재고
- 현재 재고 수량
- 제품별 재고 (선택)

### 작업 내용
- 기타 작업 사항
- 특이사항
- 메모

## 데이터 저장

모든 데이터는 `/Users/mac/clawd/memory/work-logs/` 에 날짜별로 저장됩니다:

```
work-logs/
├── 2026-01-30.json  # 오늘 기록
├── 2026-01-29.json  # 어제 기록
├── 2026-01-28.json
└── ...
```

## 통계 생성

### 일일
- 오늘의 생산/발송/재고

### 주간
- 이번 주 총 생산량
- 이번 주 총 발송 건수
- 평균 일일 생산량

### 월간
- 이달 총 생산량
- 이달 총 발송 건수
- 일별 추이

### 분기/연간
- 분기별 생산량
- 연간 발송 건수
- 트렌드 분석

## 작업일지 자동 생성

요청하시면 다음 형식으로 작업일지를 자동 생성합니다:

```
작업일지 - 2026년 1월 30일

[생산]
- 총 생산량: 50개
- 작업 시간: 오전 9시~오후 5시

[발송]
- 총 발송 건수: 30건
- 택배사: 롯데택배

[재고]
- 현재 재고: 20개
- 전일 대비: -10개

[특이사항]
- (기록된 메모)

[통계]
- 이번 주 누적 생산: 250개
- 이번 달 누적 발송: 450건
```

## 예시

### 입력
```
세미, 오늘 아침에 생산 30개 하고 
오후에 택배 25건 보냈어. 
재고는 15개 남았고 내일은 50개 더 만들어야 해
```
