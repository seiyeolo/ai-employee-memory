# 작업 일지 - 빠른 시작 가이드

## 💬 편하게 말하세요!

### 기록하기

```
세미, 오늘 생산 50개

세미, 발송 30건 했어

세미, 재고 20개 남음

세미, 오늘 생산 30개 하고 발송 15건, 재고는 25개

세미, 아침에 20개 만들고 오후에 택배 10건 보냄
```

### 확인하기

```
세미, 오늘 뭐했지?

세미, 이번 주 생산량

세미, 이달 발송 건수

세미, 작업일지 작성해줘
```

## 📊 자동으로 정리되는 것

✅ 일일 생산량  
✅ 일일 발송 건수  
✅ 현재 재고  
✅ 주간/월간 통계  
✅ 작업 일지 자동 생성  

## 🎯 핵심

**복잡한 양식 없음**  
**그냥 편하게 말하면 됨**  
**세미가 다 정리함**
