# 개인 재무 기록 시스템

## 목적
개인 지출/수입을 자동으로 기록하고 관리합니다.

## 사용 방법

### 지출 기록
```
세미, 오늘 커피 5000원
세미, ANTHROPIC USD 25.00 사용
세미, 저녁 식사 35000원
```

### 수입 기록
```
세미, 월급 입금 300만원
세미, 프리랜스 수입 50만원
```

### 확인
```
세미, 이번 달 지출은?
세미, 오늘 얼마 썼어?
세미, 2월 지출 내역
```

## 자동 카테고리
- 클라우드: AWS, Google Cloud 등
- AI API: Anthropic, OpenAI 등
- 식비: 커피, 식사 등
- 교통: 택시, 주유 등
- 기타

## 데이터 저장
`memory/finances/YYYY-MM.json` - 월별 지출/수입

## 통계
- 일별/주별/월별 지출 합계
- 카테고리별 분석
- 전월 대비 증감
