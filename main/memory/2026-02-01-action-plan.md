## 📋 STEP 1: 여자친구에게 현지 상황 파악 (오늘)

**일시:** 2026년 2월 1일 (토) - 최대한 빠르게

**확인 사항:**
- [ ] 마우이 숙소 가능 여부
  - 언니 남편 문제가 해결되었나?
  - 언니 집에 머물 수 있나?
  - 호텔/Airbnb가 필요한가?
  
- [ ] 함께 움직일 수 있는 기간
  - 2월 4일~13일 중 몇 일이나 가능?
  - 마우이에 함께 갈 수 있나?
  - 오아후에서만 만날 수 있나?
  
- [ ] 마우이 추천 정보
  - 골프장 추천
  - 관광지 추천
  - 맛집/로컬 정보

**결과에 따른 판단:**
- ✅ **마우이 가능** → A안 또는 B-1안 준비
- ❌ **마우이 불가능** → 대표에게 본토 여행 제안 + 본인은 다른 옵션 선택

## 📋 STEP 2: 대표에게 전달 (여자친구 확인 후)

**일시:** 2월 1일 저녁 또는 2월 2일

**시나리오별 전달 내용:**

### 시나리오 A: 마우이 가능한 경우
```
"대표님, 여자친구한테 확인했는데 마우이 숙소 괜찮다고 합니다.
함께 마우이 가시겠습니까? 
골프 치고 관광하기 좋은 곳이라고 합니다."
```
→ 대표 의향에 따라 A안 실행

### 시나리오 B: 마우이 불가능한 경우
```
"대표님, 마우이가 상황이 안 좋아서 어렵다고 합니다.
대신 대표님은 본토(LA나 샌프란시스코) 여행 어떠세요?
저는 하와이에서 알아서 잘 있겠습니다."
```
→ 대표 본토행 제안 + 본인은 B-2 또는 수정된 플랜

**목적:**
- 대표가 충분한 시간을 갖고 일정 재조정 가능
- 본토 여행 시 항공권 예약 시간 확보
- 서로 부담 없이 분리 가능

## 📋 STEP 3: 최종 플랜 선택

**결정 시점:** 2월 2일 (일) 저녁까지

### 경우의 수

#### 경우 1: 마우이 ✅ + 대표 동행 ✅
→ **A안 실행**
- 호놀룰루 → 마우이 2-3박 → 호놀룰루
- 마우이 항공권 즉시 예약
- 마우이 숙소 확정

#### 경우 2: 마우이 ✅ + 대표 본토행 ✅
→ **B-1안 실행 (마우이 단독 또는 여자친구와)**
- 대표는 본토 (LA/SF)
- 본인은 마우이 3-4일
- 여자친구 합류 가능하면 함께
- 호텔/Airbnb 예약

#### 경우 3: 마우이 ❌ + 대표 본토행 ✅
→ **B-2안 실행 (오아후 집중)**
- 대표는 본토
- 본인은 오아후 섬 심층 탐방
- 여자친구와 로컬 투어
- 골프, 하이킹, 스노클링

#### 경우 4: 마우이 ❌ + 대표 동행 ✅
→ **수정 A안: 다른 섬 또는 오아후**
- 대표와 함께 빅아일랜드 or 카우아이
- 또는 오아후에서 골프/관광 집중
