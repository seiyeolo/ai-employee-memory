### 상품 설명 (마켓용)
```
5인 소기업을 위한 AI 직원 조직 구축 키트.

텔레그램으로 "재고 보고해줘" 한마디면, AI 오케스트레이터가
생산관리부·물류부·영업부·콘텐츠부·재무부에 자동 위임하고
결과를 취합해서 보고합니다.

포함 내용:
✅ 6개 부서 에이전트 설정 파일
✅ 온톨로지 기반 데이터 흐름 설계
✅ 8개 업무 스킬 (재고확인, 출고처리, 주문처리 등)
✅ 23개 크론잡 자동화 세트
✅ 실패사례 가이드 (실전에서 검증된 7개 이슈+해결법)
✅ 설치부터 운영까지 단계별 가이드

만든 사람: (주)퍼티스트 — 실제로 이 시스템으로 회사를 운영 중.
```

### 타겟 고객
| 구분 | 설명 |
|------|------|
| 1차 타겟 | 1~10인 소기업 대표/총괄 (제조, 유통, 서비스) |

| 3차 타겟 | AI 에이전트 구축에 관심 있는 개발자/컨설턴트 |

## 2. AI 윤리 및 보안 가이드 (퍼티스트 버전)

### AI 윤리 7원칙 적용

| 원칙 | 퍼티스트 적용 |
|------|------------|
| 1. 생명 보호 | 제품 안전 — AS 접수 시 화재위험 감지 → 즉시 긴급 회수 |
| 2. 정서적 안전 | 고객 CS — AI임을 명확히 밝힘, 감정적 문의는 사람에게 전달 |
| 3. 관계 우선 | 에이전트는 직원의 보조자, 대체자가 아님 |
| 4. 정직성 | 재고 데이터 추정 금지 — 확인된 사실만 보고 |
| 5. 샌드박스 우선 | 외부 발송(이메일, 택배) 전 파이오니어 승인 필수 |
| 6. 인증 보관 | API키/비밀번호는 .env로 관리, 절대 평문 저장 안 함 |
| 7. 이상 징후 탐지 | 안전재고 이하 → 즉시 긴급 보고 |

### 보안 수칙

- 직원 개인정보(급여, 주민번호) 외부 노출 절대 금지
- GitHub repo는 반드시 Private
- 텔레그램 봇 토큰 노출 금지
- 거래처 정보 외부 공유 금지

## 5. FAQ (자주 묻는 질문 5가지)

**Q1. 코딩을 못 해도 되나요?**
A. 네. AGENTS.md, SKILL.md는 업무 매뉴얼을 쓰는 것과 같습니다. 코딩이 아니라 글쓰기에 가깝습니다.

**Q2. 비용이 얼마나 드나요?**
A. OpenRouter 무료 모델 활용 시 월 1~3만원 수준. 하트비트 끄기, 모델 라우팅으로 90% 절감 가능.

**Q3. 우리 회사 업종에도 적용되나요?**
A. 온톨로지(업무 흐름도)만 바꾸면 어떤 업종이든 가능합니다. 제조, 유통, 서비스, 교육 모두 적용 사례 있음.

**Q4. 에이전트가 실수하면 어떡하나요?**
A. 실수합니다. 하지만 이슈로그에 기록하면 같은 실수를 반복하지 않습니다. "사고가 나면 규칙이 생기고, 규칙이 쌓이면 에이전트가 성장합니다."

**Q5. 기존 ERP/CRM과 연동되나요?**
A. Chrome DevTools MCP로 웹 기반 시스템은 크롤링 가능. API가 있는 시스템은 직접 연동 가능.
