## Q1. OpenClaw에서 MCP 도구가 에이전트에 노출되지 않음 (#010)

## Q2. exec→Claude Code 호출 시 기존 크롬이 아닌 새 브라우저 열림 (#015)

## Q3. exec 도구 보안 승인 절차 (#016)

## Q4. session_send 실제 사용 방법
