# 구글시트 → GitHub 자동 연동 방법

## 방법 비교

| 방법 | 난이도 | 실시간 | 비용 | 추천 |
|------|-------|-------|------|------|
| **A. 구글시트 → CSV 발행 → 크론 수집** | ⭐ 쉬움 | 30분 | 무료 | ✅ 추천 |
| **B. Google Apps Script → GitHub API** | ⭐⭐ 중간 | 즉시 | 무료 | 고급 |
| **C. GitHub Actions → Sheets API** | ⭐⭐⭐ 어려움 | 스케줄 | 무료 | 대규모 |

---

## 방법 A: 구글시트 CSV 발행 + OpenClaw 크론 (추천)

### 1단계: 구글시트를 CSV로 발행
1. 구글시트 열기
2. 파일 → 공유 → 웹에 게시
3. 형식: CSV 선택
4. "게시" 클릭 → CSV URL 복사

URL 형식:
```
https://docs.google.com/spreadsheets/d/{시트ID}/export?format=csv&gid={탭번호}
```

### 2단계: 수집 스크립트 작성
```bash
#!/bin/bash
# ~/clawd/scripts/sync-gsheet.sh
# 구글시트 → 로컬 CSV → GitHub 동기화

SHEET_URL="구글시트_CSV_URL_여기"
OUTPUT="/Users/mac/clawd-production/data/일일업무일지.csv"

curl -sL "$SHEET_URL" -o "$OUTPUT"
echo "[gsheet-sync] $(date '+%H:%M') - 구글시트 동기화 완료"
```

### 3단계: OpenClaw 크론 등록
```bash
openclaw cron add \
  --name "구글시트동기화" \
  --schedule "*/30 * * * *" \
  --message "exec bash ~/clawd/scripts/sync-gsheet.sh"
```

→ 30분마다 자동으로 구글시트 데이터를 CSV로 내려받아 GitHub에 동기화됨

## 방법 B: Google Apps Script → GitHub 직접 푸시

### 구글시트 측 스크립트 (확장 프로그램 → Apps Script)
```javascript
function pushToGitHub() {
  var sheet = SpreadsheetApp.getActiveSheet();
  var data = sheet.getDataRange().getValues();

  // CSV 변환
  var csv = data.map(row => row.join(',')).join('\n');

  // GitHub API로 파일 업데이트
  var token = PropertiesService.getScriptProperties().getProperty('GITHUB_TOKEN');
  var repo = 'seiyeolo/ai-employee-memory';
  var path = 'production/data/일일업무일지.csv';

  // 기존 파일 SHA 가져오기
  var getUrl = 'https://api.github.com/repos/' + repo + '/contents/' + path;
  var getResp = UrlFetchApp.fetch(getUrl, {
    headers: { 'Authorization': 'token ' + token }
  });
  var sha = JSON.parse(getResp.getContentText()).sha;

  // 파일 업데이트
  var putUrl = getUrl;
  UrlFetchApp.fetch(putUrl, {
    method: 'put',
    headers: {
      'Authorization': 'token ' + token,
      'Content-Type': 'application/json'
    },
    payload: JSON.stringify({
      message: 'gsheet sync: ' + new Date().toISOString(),
      content: Utilities.base64Encode(csv, Utilities.Charset.UTF_8),
      sha: sha
    })
  });
}

// 트리거 설정: 편집 시 또는 시간 기반
function createTrigger() {
  ScriptApp.newTrigger('pushToGitHub')
    .timeBased()
    .everyHours(1)
    .create();
}
```

### 설정 순서
1. 구글시트 → 확장 프로그램 → Apps Script
2. 위 코드 붙여넣기
3. 스크립트 속성에 `GITHUB_TOKEN` 추가 (GitHub Personal Access Token)
4. `createTrigger()` 실행 → 1시간마다 자동 푸시

        ↑                                ↓
  사용자 직접 입력              OpenClaw 에이전트가 읽기

## 주의사항
- 구글시트 CSV 발행 URL은 **누구나 접근 가능** → 민감 데이터 주의
- 방법 B(Apps Script)가 보안상 더 안전 (토큰 기반)
- 기존 `puttist-inventory-system` 리포에 이미 Apps Script 기반 구축 경험 있음
