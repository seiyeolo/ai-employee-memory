# OpenClaw AI 직원 조직 구축 프로젝트

## 프로젝트 개요
- **날짜:** 2026년 3월 28~29일
- **참여자:** 오세열 파이오니어 (총괄이사) + Claude Code (Opus 4.6)
- **목적:** OpenClaw를 활용하여 (주)퍼티스트의 회사 업무를 AI 직원 조직으로 자동화
- **출발점:** OpenClaw 설치는 되어있으나 "어떻게 일을 시켜야 하는지 모르는" 상태

## 핵심 문제
1. OpenClaw가 설치되어 있지만 효율적으로 활용하지 못함
2. 에이전트 1개(main)만 운영, 모든 업무가 한 곳에 몰림
3. 크론 작업 12개 중 5개 실패 (Anthropic 크레딧 부족)
4. 스킬 72개 중 43개 차단
5. 회사 관리 시스템(인사, 재고, AS 등)이 체계화되지 않음

## 해결 접근법
NotebookLM의 "OpenClaw 실전 가이드" + OpenClaw School 교육자료를 참고하여:
- 비서 AI → 직원 AI 전환
- 사수-부사수(오케스트레이터-부서장) 구조 도입
- session_send를 통한 내부 통신 체계 구축
- GitHub를 통한 장기 기억(Second Brain) 구현

## 성과
- 에이전트 1개 → 5개 부서 조직 구축
- 크론 5개 실패 → 전체 정상화 + 10개로 확장
- 인사관리 체계 신규 구축
- 외부 시스템 연동 (Google Sheets AS, 스마트스토어)
- Knowledge Base 구축 (제품 전문 지식)
- 비용 최적화 (Anthropic → OpenAI + 로컬 모델 혼합)
