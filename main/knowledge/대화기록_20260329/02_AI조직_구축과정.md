## 3단계: 에이전트 생성

### 명령어
```bash
openclaw agents add production --workspace ~/clawd-production --model anthropic/claude-sonnet-4-5 --non-interactive
openclaw agents add logistics --workspace ~/clawd-logistics --model anthropic/claude-sonnet-4-5 --non-interactive
openclaw agents add sales --workspace ~/clawd-sales --model anthropic/claude-sonnet-4-5 --non-interactive
openclaw agents add content --workspace ~/clawd-content --model anthropic/claude-sonnet-4-5 --non-interactive
```

### 각 부서 워크스페이스 구성
- AGENTS.md: 부서 역할, 담당 업무, 보고 형식, 금지 사항
- SOUL.md: 성격, 말투, 보고 원칙
- IDENTITY.md: 이름, 소속, 상급자
- TOOLS.md: 로컬 환경 설정, 외부 시스템 연결 정보
- knowledge/: 전문 지식 파일
- data/: 업무 데이터

## 5단계: Anthropic 크레딧 복구

### 문제
auth-profiles.json의 `disabledUntil` 타이머가 9시간 남아있어 크레딧 충전 후에도 차단

### 해결
```json
"disabledUntil": 0,
"disabledReason": null
```
직접 리셋 후 게이트웨이 재시작

### 검증
일일발송량집계 크론 수동 실행 → 성공 (claude-sonnet-4-5, 텔레그램 전송 완료)
