# 인사관리 체계 구축 기록

## 배경
- 5인 사업장(대표1 + 총괄이사1 + 상주1 + 시간제3)
- 인사관리 문서가 전혀 없는 상태
- 5인 이상 사업장 → 근로기준법 대부분 적용

## 5인 사업장 법적 요건 (핵심)
- 부당해고 금지 + 해고 시 서면 통지 필수
- 연장/야간/휴일 50% 가산수당 필수
- 연차유급휴가 필수 (15~25일)
- 직장 내 괴롭힘 금지 적용
- 근로계약서 미교부 시 500만원 벌금
- 임금명세서 미교부 시 500만원 과태료
- 주 52시간 제한 적용

## 시간제 근로자 특별 주의사항
- 근로계약서에 근로일 및 근로일별 근로시간 서면 명시 (과태료 500만원)
- 주 15시간 이상: 주휴수당, 연차, 4대보험 적용
- 주 15시간 미만: 주휴수당/연차/퇴직급여 미적용
- 산재보험은 무조건 적용 (시간 무관)
