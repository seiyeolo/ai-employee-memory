## 실사 완료 후 처리

1. 이 표를 Claude Code에게 전달 → 시스템 데이터 일괄 업데이트
2. 사진 파일을 `clawd-production/data/photos/`에 정리
3. 차이 발생 품목은 원인 분석
4. 신규 발견 품목은 품번 부여 후 부품마스터.csv에 추가
5. **상품 수량 → 영업마케팅부에 공유 (온톨로지)**
6. 한 달간 데이터 축적 후 → 대표 보고
