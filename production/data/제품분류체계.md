## 3. 일본형 본체 (슬리브 씌운 것) - 4가지 타입

### 타입 1: 배터리 O + 레이저 O
- 충전배터리: 있음
- 레이저 에이밍: 있음
- **풀 사양**

### 타입 2: 배터리 X + 레이저 X
- 충전배터리: 없음
- 레이저 에이밍: 없음
- **베이직 사양**

### 타입 3: 배터리 O + 레이저 X
- 충전배터리: 있음
- 레이저 에이밍: 없음
- **배터리 전용**

### 타입 4: 배터리 X + 레이저 O
- 충전배터리: 없음
- 레이저 에이밍: 있음
- **레이저 전용**

## 참고 사항

- **고급형 = 한국형 풀 사양 (배터리 O + 레이저 O)**
- **일본형 = 슬리브 씌운 상태에서 사양 선택 가능**
- **종이슬리브 = 포장재 (영문/일어 등 버전 구분)**
