> **목적:** "AI 온라인 회사" B2B 서비스 기획을 위한 시장조사
> **사용처:** Google NotebookLM 딥리서치

## 운영 가이드

### 노트북 분리 전략
단계별로 **별도 노트북을 생성**하여 딥리서치를 실행합니다.

| 노트북 이름 | 단계 | 소스 구성 |
|------------|------|----------|
| `셀러리서치-1-시장규모` | 1단계 | 통계/IR 자료 + 산업 리포트 |
| `셀러리서치-2-페인포인트` | 2단계 | 1단계 결과물 + 커뮤니티/설문 자료 |
| `셀러리서치-2.5-시간경제학` | 2.5단계 | 1+2단계 결과물 + 셀러 루틴/자동화 사례 |
| `셀러리서치-3-솔루션기회` | 3단계 | 1+2+2.5단계 결과물 + SaaS/해외 벤치마크 |
| `셀러리서치-4-페르소나기획` | 4단계 | 전체 결과물 + 인터뷰/사례 연구 |

**왜 분리하는가:**
- 각 노트북의 소스가 해당 단계에 집중 → 환각 감소, 분석 정밀도 상승
- 이전 단계 결과물을 "검증된 소스"로 다음 노트북에 추가 → 누적 품질 보장
- 단계별로 소스 교체/추가가 자유로움 → 부족하면 소스 보강 후 재실행 가능

### 데이터 수집 제외 조건 (전 단계 공통)

다음 데이터는 분석에서 **제외**하거나 신뢰도를 낮춰 처리해 주세요:

```
⛔ 제외 대상:
- 2022년 이전의 시장 데이터 (코로나 특수로 왜곡됨)
- 특정 솔루션/플랫폼의 자사 홍보성 보도자료 (객관성 결여)
- 출처가 불명확한 커뮤니티 루머 또는 익명 후기
- 해외(미국/중국) 데이터를 한국에 그대로 대입한 추정치
- 쿠팡 로켓배송/직매입 부문 (셀러 마켓플레이스와 구조가 다름)
- 도매/B2B 거래 데이터 (우리 타겟은 B2C 셀러)
- 명품/럭셔리 카테고리 (일반 셀러와 비즈니스 모델이 다름)
- 드롭쉬핑 전용 셀러 데이터 (운영 구조가 근본적으로 다름)

⚠️ 신뢰도 낮춤 처리:
- 플랫폼 자체 발표 셀러 수 (비활성 계정 포함 가능성)
- 단일 설문조사 기반 추정 (표본 편향 가능성 명시)
- 1년 이상 된 가격/수수료 정보 (수시 변경됨)
```

## 1단계: 시장 규모 및 셀러 현황 파악

### 노트북: `셀러리서치-1-시장규모`

### 추천 소스
- 통계청 온라인쇼핑 동향 보고서 (2023~2026)
- 공정거래위원회 오픈마켓 관련 보도자료
- 각 플랫폼 공식 IR 자료 또는 뉴스 기사 (쿠팡, 네이버 스마트스토어, 11번가, G마켓/옥션, 위메프, 티몬)
- 이커머스 산업 리포트 (코리안클릭, 닐슨코리아, 오픈서베이 등)
- 고용노동부 자영업 전환 통계

### 프롬프트

```
대한민국 오픈마켓(이커머스 마켓플레이스) 셀러 생태계의 전체 그림을 파악하고 싶습니다.
이것은 "AI 기반 셀러 지원 서비스" 기획을 위한 시장 규모 파악이 목적입니다.

⛔ 다음 데이터는 제외해 주세요:
- 2022년 이전 데이터 (코로나 특수 왜곡)
- 쿠팡 로켓배송/직매입 부문 (셀러 마켓플레이스가 아님)
- 도매/B2B, 명품/럭셔리, 드롭쉬핑 전용 셀러
- 플랫폼 자사 홍보성 보도자료 (객관성 결여)
- 출처 불명확한 커뮤니티 루머

다음 항목을 중심으로 조사해 주세요:

1. **셀러 규모 추정**
   - 주요 플랫폼별(쿠팡 마켓플레이스, 네이버 스마트스토어, 11번가, G마켓/옥션) 등록 셀러 수와 활성 셀러 수
   - 전체 오픈마켓 셀러를 자영업자/개인사업자, 소규모 법인, 부업 셀러로 구분했을 때 각 비율 추정
   - 최근 3년간(2023~2026) 셀러 수 증감 추이

2. **신규 진입 셀러 동향**
   - AI/자동화로 인한 실직자의 셀러 전환 추이 (2024~2026)
   - 신규 셀러 진입 후 1년 생존율, 3년 생존율
   - 셀러가 외주/도구/솔루션에 지출하는 월평균 비용

3. **시장 구조**
   - 플랫폼별 거래액(GMV) 규모와 시장 점유율
   - 셀러 1인당 평균 월 매출 규모 (평균값 + 중앙값)
   - 멀티채널 셀러(2개 이상 플랫폼 동시 입점) 비율

4. **셀러 인구통계**
   - 연령대, 성별, 지역 분포
   - 전업 셀러 vs 부업 셀러 비율
   - 평균 운영 기간(셀러 생존율 곡선)

모든 수치에는 출처와 시점을 명시하고, 데이터가 불확실한 경우 추정의 근거를 함께 제시해 주세요.
최종 결과는 표와 차트 형태로 정리해 주세요.
```

## 2단계: 셀러 페인포인트 심층 분석

### 노트북: `셀러리서치-2-페인포인트`

### 소스 구성
- **필수**: 1단계 딥리서치 결과물 (소스로 추가)
- 셀러 커뮤니티 게시글 (네이버 카페: 셀러노트, 스마트스토어 카페 등)
- 셀러 대상 설문조사 보고서
- 이커머스 업계 뉴스 기사 (플랫폼 정책 변경 관련)
- 셀러 유튜브 채널의 고충 관련 콘텐츠 정리본

### 프롬프트

```
대한민국 오픈마켓 자영업 셀러들이 일상적으로 겪는 페인포인트(고충)를 구조화해서 분석해 주세요.
"AI가 셀러의 온라인 회사를 만들어준다"는 서비스를 기획하기 위한 기초 리서치입니다.

⛔ 다음은 제외해 주세요:
- 대형 법인 셀러(직원 10명 이상)의 고충 (우리 타겟이 아님)
- 특정 솔루션의 홍보성 사례 (객관성 결여)
- 2022년 이전 정책/수수료 기준 (현재와 다름)
- 해외 셀러 사례를 한국에 그대로 대입한 분석

다음 카테고리별로 페인포인트를 정리해 주세요:

1. **운영 효율성**
   - 상품 등록/관리 (멀티채널 상품 정보 동기화, 카테고리 매핑 등)
   - 주문 처리 및 배송 (출고, 반품, 교환 처리)
   - 재고 관리 (다채널 재고 연동, 과잉재고/품절 문제)
   - CS 대응 (고객 문의, 악성 리뷰, 분쟁 처리)

2. **수익성 압박**
   - 플랫폼 수수료 구조와 부담
   - 광고비/마케팅비 증가 (CPC 상승, 광고 없이는 노출 불가)
   - 가격 경쟁 심화 (최저가 전쟁, 마진 축소)
   - 정산 주기 및 현금흐름 문제

3. **플랫폼 종속성**
   - 알고리즘 변경에 따른 매출 변동
   - 플랫폼 정책 일방 변경 리스크
   - 자체 브랜딩/고객 데이터 확보의 어려움
   - 플랫폼 자체 PB상품과의 경쟁

4. **역량 부족**
   - 상품 사진/상세페이지 제작 역량
   - 데이터 분석 및 트렌드 파악
   - 세무/회계/법률 지식 부족
   - 디지털 마케팅(SEO, SNS, 라이브커머스) 역량

5. **외부 환경**
   - 해외 직구/알리익스프레스·테무 등 C2C 플랫폼의 국내 진출
   - 물류비 상승
   - 인력 채용 및 관리 (1인 사업 한계)

6. **시간 소비 분석** ⭐ (핵심)
   - 셀러가 하루 중 각 업무에 투입하는 시간 비율
     (상품등록 _%, 상세페이지 _%, CS _%, 발송 _%, 마케팅 _%, 재무/세무 _%)
   - "시간을 돌려받으면 가장 하고 싶은 것"이 무엇인지
   - 현재 자동화하고 있는 업무 vs 여전히 수동인 업무 비율

각 페인포인트에 대해:
- 심각도를 상/중/하로 평가
- 영향받는 셀러 비율 추정
- 구체적 사례나 셀러 커뮤니티의 실제 목소리 인용
- 현재 시장에 나와 있는 부분적 해결책(있다면)

결과를 심각도 순으로 정렬하고, 가장 큰 페인포인트 TOP 5를 별도로 요약해 주세요.
```

## 2.5단계: 셀러의 시간 경제학 분석 ⭐ (신규)

### 노트북: `셀러리서치-2.5-시간경제학`

### 소스 구성
- **필수**: 1단계 + 2단계 딥리서치 결과물
- 셀러 하루 루틴 관련 블로그/유튜브 콘텐츠
- 이커머스 자동화 도구 비교 기사
- 셀러 시간관리 관련 설문/인터뷰 자료

### 프롬프트

```
오픈마켓 셀러의 "시간"을 경제학적으로 분석해 주세요.
우리 서비스의 핵심 가치 제안은 "시간을 돌려주는 것"이므로, 이 분석이 가격 책정과 마케팅 메시지의 근거가 됩니다.

⛔ 다음은 제외해 주세요:
- 직원 5명 이상 운영하는 대형 셀러 (이미 조직이 있음)
- 월 매출 100만원 미만 취미 셀러 (지불 의향 낮음)
- 자동화 도구의 홍보성 시간절약 수치 (과장 가능성)

다음 항목을 분석해 주세요:

1. **전업 셀러(월 매출 500만~3,000만)의 일일 업무 타임라인**
   - 06:00~24:00 중 각 업무별 소요 시간을 시간표로 작성
   - 업무 카테고리: 소싱/매입, 상품등록, 상세페이지 제작, 사진촬영, 
     주문처리, 포장/발송, CS응대, 마케팅/SNS, 재고관리, 세무/정산, 기타
   - 각 업무의 특성 분류:
     ① 반드시 사람이 해야 하는 것 (소싱 판단, 품질 확인 등)
     ② AI/자동화로 대체 가능한 것 (상세페이지, CS 초벌, 재고 알림 등)
     ③ AI가 보조하면 절반으로 줄일 수 있는 것 (마케팅 기획, 세무 등)

2. **시간의 경제적 가치 환산**
   - 셀러의 시간당 기회비용 계산 (월 매출 기준)
   - 자동화 가능 업무의 총 시간 × 시간당 가치 = 월간 절약 가능 금액
   - "직원 1명 고용 비용(최저임금 월 209만원)" 대비 절약 가능 금액 비교

3. **현재 자동화 도구 사용 현황과 한계**
   - 셀러들이 실제로 쓰는 도구 목록과 월 비용
   - 도구를 써도 여전히 수동으로 남는 업무 영역 (= 우리의 기회)
   - 도구 학습에 드는 시간 비용 (새 도구를 배우는 것 자체가 부담)

4. **시간 절약의 파급 효과**
   - 시간이 돌아오면 셀러가 할 수 있는 것들
     (신규 상품 소싱, 채널 확장, 브랜딩, 라이프스타일 개선 등)
   - 시간 절약 → 매출 증대로 이어지는 선순환 구조 분석

최종 결과를 다음 형태로 정리:
- "AI 조직이 하루 X시간을 돌려주면, 월 Y만원의 가치 = 직원 Z명분" 공식
- 이 공식이 우리 서비스 가격(월 구독료)의 정당성 근거가 될 수 있도록
```

## 3단계: 셀러 니즈 및 솔루션 기회 도출

### 노트북: `셀러리서치-3-솔루션기회`

### 소스 구성
- **필수**: 1+2+2.5단계 딥리서치 결과물 전체
- 셀러 대상 SaaS 서비스 소개 자료 (사방넷, 셀러허브, 플레이오토, 셀러노트 등)
- 해외 셀러 툴 사례 (Jungle Scout, Helium 10, Shopify 앱스토어 인기 앱 등)
- 이커머스 SaaS 투자 동향 기사

### 프롬프트

```
이전 단계에서 도출한 셀러 페인포인트와 시간 경제학 분석을 바탕으로,
셀러들의 미충족 니즈(unmet needs)와 새로운 서비스 기회를 분석해 주세요.

⛔ 다음은 제외해 주세요:
- 이미 레드오션인 단순 가격비교/최저가 알림 서비스
- 물류/풀필먼트 자체를 대행하는 사업 (우리 범위 아님)
- 플랫폼 자체를 만드는 사업 (마켓플레이스 창업)
- 하드웨어 기반 솔루션 (바코드 스캐너, 포장기계 등)

1. **니즈 맵핑**
   각 주요 페인포인트에 대해:
   - 셀러가 실제로 원하는 이상적인 상태(To-Be)는 무엇인가?
   - 현재 해결책(As-Is)은 무엇이고 왜 불충분한가?
   - 해결 시 셀러에게 돌아가는 가치(시간 절약, 비용 절감, 매출 증대 등)는?

2. **기존 솔루션 경쟁 분석**
   현재 한국 시장에 있는 셀러 대상 서비스를 분류해 주세요:
   - 통합 관리 솔루션 (사방넷, 셀러허브, 플레이오토 등)
   - 특화 솔루션 (상세페이지 제작, 리뷰 관리, 가격 모니터링 등)
   - 각 솔루션의 강점, 약점, 가격대, 시장 점유율
   - 커버하지 못하는 공백(gap) 영역

3. **"도구 vs 조직" 패러다임 비교** ⭐
   - 기존 셀러 도구(사방넷, 플레이오토)는 "도구를 준다" (셀러가 직접 써야 함)
   - AI 에이전트 기반 솔루션은 "조직을 준다" (셀러는 CEO처럼 지시만 함)
   - 이 차이가 만드는 가치:
     ① 학습 비용: 도구를 배우는 시간 vs 자연어로 지시하는 시간
     ② 시간 절약 효과: 부분 자동화 vs 업무 전체 위임
     ③ 확장성: 도구를 추가해야 함 vs 에이전트에 스킬을 추가하면 됨
   - 셀러 입장에서 어떤 것이 더 매력적인지 근거와 함께 분석

4. **해외 벤치마크**
   아마존/쇼피파이 생태계에서 성공한 셀러 툴 중 한국에 아직 없거나 부족한 것:
   - 해외에서는 해결됐지만 한국에서는 미해결인 문제
   - 한국 시장 특수성(네이버 생태계, 카카오 연동 등)을 고려한 현지화 기회

5. **솔루션 기회 우선순위**
   다음 기준으로 기회를 평가하고 우선순위를 매겨 주세요:
   - 시장 크기 (TAM/SAM/SOM 추정)
   - 페인포인트 심각도 (셀러의 지불 의향)
   - 기술적 구현 난이도
   - 경쟁 강도
   - 수익 모델 가능성 (SaaS 구독, 수수료, 프리미엄 등)

결과를 기회 매력도 순으로 정렬한 매트릭스로 정리하고,
TOP 3 기회에 대해서는 간략한 서비스 컨셉까지 제안해 주세요.
```

## 4단계: 타겟 셀러 페르소나 및 서비스 기획 방향 정리

### 노트북: `셀러리서치-4-페르소나기획`

### 소스 구성
- **필수**: 1~3단계 딥리서치 결과물 전체
- 실제 셀러 인터뷰 또는 사례 연구 자료 (있다면)

### 프롬프트

```
지금까지의 리서치 결과를 종합하여, "AI 온라인 회사" 서비스 기획에 바로 활용할 수 있는 최종 정리를 해주세요.

⛔ 다음은 제외해 주세요:
- 비현실적인 이상향 (모든 문제를 한번에 해결하는 만능 솔루션)
- 기술 중심 사고 (셀러는 기술이 아닌 결과에 관심이 있음)
- 대기업 셀러 페르소나 (우리 타겟이 아님)

1. **타겟 셀러 페르소나 (4개)**
   각 페르소나별로:
   - 이름(가상), 나이, 셀러 유형(전업/부업), 취급 카테고리
   - 월 매출 규모, 입점 플랫폼, 운영 기간
   - 하루 일과 타임라인 (셀러로서의 루틴)
   - 가장 큰 3가지 고충
   - "이런 게 있으면 좋겠다"는 바람
   - 현재 사용 중인 툴과 월 지출 비용
   - **페르소나 유형 반드시 포함:**
     ① 전업 파워셀러 (월 매출 2,000만+)
     ② 성장기 셀러 (월 매출 500~1,500만)
     ③ 신규 진입 셀러 (퇴직 후 6개월 이내)
     ④ 부업 셀러 (직장인, 주부 등)

2. **핵심 인사이트 요약**
   - 셀러 시장에서 가장 큰 미충족 니즈 3가지
   - 지금 이 시장에 진입하기 좋은 이유 (타이밍)
   - 진입 시 주의해야 할 리스크 TOP 3

3. **신뢰 장벽 분석** ⭐
   - 셀러가 AI에 사업 데이터(재고, 매출, 고객정보)를 맡기는 것에 대한 저항감 수준
   - "AI가 내 고객에게 직접 대응한다"에 대한 수용도
   - 신뢰를 얻기 위해 필요한 최소 조건
     (예: 인간 승인 게이트, 행동 로그 투명 공개, 되돌리기 기능 등)
   - 페르소나별 신뢰 장벽 차이 분석

4. **경쟁 우위 차별화 포인트**
   - "도구를 주는 것"(기존 SaaS) vs "AI 조직을 만들어주는 것"(우리)의 포지셔닝
   - "직원 1명 월급보다 싼 AI 조직" 메시지의 설득력 검증
   - 기존 솔루션에서 우리로 전환하게 만들 핵심 트리거

5. **Next Step 제안**
   - 검증이 필요한 핵심 가설 3~5개
   - 추천 검증 방법 (셀러 인터뷰, 프로토타입 테스트, 랜딩페이지 실험 등)
   - MVP 구성 시 반드시 포함해야 할 핵심 기능 3가지
   - 가격 모델 초안 (구독 tier 제안)

최종 결과물은 사업 기획서나 투자 피칭 자료의 "시장 분석" 섹션으로 바로 활용할 수 있는 수준으로 정리해 주세요.
```
