## 페인포인트 TOP 5 (심각도 상)

### 1. 상세페이지/이미지 제작의 고통 (건당 2~3일, 외주 10~30만원)
- **심각도: 상** | 영향 85%+ 셀러
- 포토샵 능력 부재 → 중국어 지우기, 누끼 따기, 상세페이지 구성에 엄청난 시간
- 기존 해결책: 드랩아트(AI), 캔바, 미리캔버스 — 부분적 해결만
- **AI 조직 기회:** 제품 사진만으로 상세페이지 자동 생성

### 2. 파편화된 다채널 수기 업무 (하루 2~3시간 증발)
- **심각도: 상** | 영향 80~90% 셀러
- 각 마켓 로그인 → 주문 수집 → 엑셀 복붙 → 송장 수기 입력 → 포장
- "보통 100개 정도 사이트에 로그인 해야 하는데 매일 엄청난 시간 로스"
- 기존 해결책: 사방넷, 이지어드민 — 쓰더라도 엑셀 수작업 여전
- **AI 조직 기회:** 모든 채널 발주/송장/재고 백그라운드 통합 연동

### 3. 1인 셀러 CS 감정노동 + 매출 하락
- **심각도: 상** | 영향 90%+ 셀러
- "혼자서 운영하는 사장님이라면 고객 응대에 신경 쓸 겨를도 없어요"
- 응답 지연 → 스토어 노출/신뢰도 하락 → 매출 감소 악순환
- 기존 해결책: 카카오 알림톡, 채널톡 — 반복 질문만 커버
- **AI 조직 기회:** AI CS 에이전트가 배송/반품/교환 실시간 자동 대응

### 4. 세무/회계 스트레스 (영수증 정리 + 마진 계산 불가)
- **심각도: 상** | 영향 90%+ 셀러
- "돈은 버는데 남는 게 없다는 막연한 고민" — 엑셀로 마진 계산 불가
- 세금 신고(5월, 1월, 7월)마다 큰 압박, 합법적 환급 놓침
- 기존 해결책: 캐시노트, 자비스, 삼쩜삼 — 신고만, 경영 판단은 못 해줌
- **AI 조직 기회:** 카드 연동 자동 장부 + 마진율 실시간 계산 + 절세 알림

### 5. 최저가 소싱 이미지 검색 노동 (마진 35~40% 사수 전쟁)
- **심각도: 상** | 영향 90%+ 셀러
- 알리/테무 공세 속에서 타오바오 이미지 검색으로 2~3천원이라도 싼 소스 찾기
- 기존 해결책: 크롬 확장프로그램 이미지 검색 — 수동 비교 필요
- **AI 조직 기회:** 이미지/URL 입력 → AI가 최저가 소싱처 + 예상 마진 자동 계산

## ⭐ 시간 소비 분석

### 셀러 하루 시간 분배 (추정)
- **단순 반복 루틴: 하루 2~3시간** (주문수집, 송장, 포장, 엑셀)
- **상세페이지 제작: 건당 2~3일** (가장 큰 시간 블랙홀)
- **CS 응대: 수시 (집중 업무 방해 요소)**
- **세무/정산: 월 2~3일 집중** (신고 기간 극심)

### 자동화 현황
```
완전 자동화:  거의 없음
부분 자동화:  사방넷/이지어드민으로 일부 주문 처리
여전히 수동:  상세페이지, CS, 엑셀 합배송, 세무, 소싱 가격 비교
              → 이 영역이 "AI 조직" 서비스의 핵심 타겟
```

### "시간을 돌려받으면 하고 싶은 것"
→ **상품 기획, 마케팅, 브랜딩, 고객 소통** (매출 창출 핵심 영역)
→ 즉, 셀러는 "CEO로서의 일"을 하고 싶지만, "직원이 할 일"에 파묻혀 있음
