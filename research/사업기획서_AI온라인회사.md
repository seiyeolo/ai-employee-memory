## 1. Executive Summary

**한 문장:** 오픈마켓 셀러에게 "AI 직원 조직"을 통째로 제공하여, 하루 3시간 수작업을 1시간으로 줄이고 나머지는 휴대폰 승인만으로 운영 가능하게 만드는 B2B SaaS.

**핵심 차별점:** 기존 솔루션(사방넷, 플레이오토)은 **"도구를 준다"** → 우리는 **"AI 조직을 만들어준다"**

**비유:** 사방넷 = 운전을 직접 해야 하는 차 / 우리 = 기사가 딸려오는 차

### TAM/SAM/SOM
- **TAM:** 활성 셀러 50~70만 × 평균 월 10만원 = **연 6,000~8,400억원**
- **SAM:** 월 매출 500만~3,000만 셀러 10~15만명 = **연 1,800억원**
- **SOM (1년차):** 1,000명 × 월 10만원 = **연 12억원**

### 진입 타이밍
1. 폐업 100만 시대 → 생존 도구 수요 급증
2. 부업자 22% 증가 → 시간 없는 셀러 급증
3. 생성형 AI 성숙 → "AI 조직" 기술적으로 가능해진 변곡점
4. 테무/알리 공세 → 운영 효율로만 생존 가능

## 3. 고객 분석

### 타겟 페르소나 (듀얼 타겟)

**주 타겟 ①: 이성장형 (성장기 셀러)**
- 32세, 전업, 의류/패션, 월 1,000만원, 스마트스토어+에이블리
- 상세페이지 1건에 2~3일 → 신상품 업로드 병목
- "디자인과 CS를 맡길 가성비 좋은 직원"을 원함
- 현재 월 5~10만원 지출

**주 타겟 ②: 최부업형 (부업 셀러)**
- 41세, 직장인 투잡, 해외 구매대행, 월 300만원, 쿠팡+11번가
- 퇴근 후 밤10시~새벽1시 작업, 하루 3시간이 한계
- "본업 중 쇼핑몰이 알아서 돌아가는 것"을 원함
- 현재 월 10~15만원 지출

### 핵심 가설
> **"셀러는 CEO로서의 일(기획, 마케팅, 브랜딩)을 하고 싶지만, 직원이 할 일(송장, CS, 엑셀, 세무)에 파묻혀 있다."**

|------|-----------|-------------|-------------------|------|
| 1 | 상세페이지 제작 | 건당 2~3일, 외주 10~30만원 | **URL → 3분 자동생성 → 승인** | 99%↓ |
| 2 | 다채널 수기업무 | 하루 2~3시간 (100개 사이트 로그인) | AI가 백그라운드 통합 처리 | 100%↓ |

| 5 | 최저가 소싱 | 타오바오 이미지 검색 노동 | AI 최저가 소싱처 + 마진 자동 계산 | 70%↓ |

## 5. 경쟁 분석

### 기존 솔루션의 한계

| 솔루션 | 유형 | 가격 | 공백 |
|--------|------|------|------|
| 사방넷 | 도구 (Back-office) | 월 25만원 | 상세페이지/마케팅/분석 미지원 |
| 플레이오토 | 도구 (Back-office) | 월 15만원 | 학습 곡선 높음, 직원교육 필요 |
| 이지어드민 | 도구 (Back-office) | - | 연동 채널 50개 한계 |
| 드랩아트 | 특화 (이미지만) | - | 채널 배포/CS/세무 미지원 |

### 해외 격차
- **Helium 10, Jungle Scout:** AI 리스팅, PPC 최적화, 소싱 검증 → 한국에 없음
- 한국은 Back-office만 발달, **Front-office 지능형 서비스 부재**

### 우리의 포지셔닝

```
사방넷/플레이오토  =  "도구" (Back-office)
Helium 10         =  "성장 엔진" (Front-office)  
우리              =  "AI 조직" (Full-stack) = 도구 + 성장 엔진 + 팀
```

| 비교 | 기존 (도구) | 우리 (AI 조직) |
|------|-----------|---------------|
| 작동 방식 | 셀러가 직접 클릭/업로드 | "등록해줘" 자연어 지시 |
| 학습 비용 | 매뉴얼 수십 장 | **제로** |
| 시간 절약 | 선형적 (N번→1번) | **기하급수적** (개입 자체 소거) |
| 확장성 | 직원 추가 필요 | AI 자동 스케일링 |
| 비용 | 월 15~25만원 + 직원 인건비 | **월 4.9~19.9만원 (직원 1/10)** |

## 6. 제품 전략

### 핵심 자산: 3종 엔진 (이미 60~80% 완성)

| 엔진 | 위치 | 역할 | 완성도 |
|------|------|------|--------|
| **detailpages_2.0** | `~/iCloud/상세페이지/detailpages_2.0` | 상세페이지 자동생성 (7개 AI Provider, 전환율 규칙) | 80% |
| **puttist_marketing_team** | `~/iCloud/puttist_marketing_team` | 멀티플랫폼 콘텐츠 동시 생성 (뉴스레터/블로그/쇼츠/SNS) | 70% |
| **odin-studio** | `~/iCloud/odin-studio` | 사물 의인화 Pixar 영상 7단계 파이프라인 (다국어) | 60% |

→ 3종 엔진을 **콘텐츠 에이전트 + 마케팅 에이전트**의 핵심 엔진으로 즉시 통합 가능

### 킬러 기능: "상품이 스스로 말한다" (경쟁사 불가능)
```
셀러: "이 텀블러 마케팅 해줘"
  → odin-studio: 텀블러를 Pixar 캐릭터로 의인화
  → "안녕! 나는 40도를 12시간 지키는 텀블러야 🥤" 15초 쇼츠
  → puttist_marketing_team: 동시에 카드뉴스 + 블로그 + SNS 포스트 생성
  → detailpages_2.0: 상세페이지도 자동 생성
  → 셀러는 승인만 → 전 채널 자동 배포
```
이것은 사방넷/플레이오토/Helium 10 **어디에도 없는 기능**.

### MVP 필수 기능 3가지

**① AI 상세페이지 에이전트 (detailpages_2.0 기반)**
- 소싱 URL 입력 → 이미지 크롭/번역/배치 + 마케팅 카피 자동 생성
- 셀러는 미리보기 확인 → "승인" 탭 → 채널 자동 배포
- *이미 80% 구현됨*

**② CS 답변 초안 생성기 (Human-in-the-loop)**
- 다채널 고객 문의 수집 → AI가 규정 기반 최적 답변 생성
- 대기 상태로 제안 → 셀러 원클릭 승인 → 자동 발송
- "완전 자동" 아닌 "승인 후 발송" = 도입률 3배↑

**③ 자연어 데이터 랩**
- "오늘 쿠팡 마진 얼마?" → 텍스트/표로 즉답
- 복잡한 대시보드 대신 대화형 인터페이스
- 박퇴직형(디지털 문해력 낮은 셀러)도 즉시 사용 가능

### 기술 아키텍처

```
셀러 (카카오톡/웹)
  → Paperclip (회사 = 오케스트레이션 + 인박스 결재 + 예산 통제)
    → OpenClaw (직원 = 실무 실행 엔진)
      ├── 콘텐츠 에이전트 (detailpages_2.0 엔진)
      ├── 마케팅 에이전트 (puttist_marketing + odin-studio)
      ├── CS 에이전트 (채널 API 연동)
      ├── 재무 에이전트 (자동 장부/마진)
      └── 소싱 에이전트 (가격 비교/마진 시뮬)
    → NemoClaw (보안관 = 위험 작업 차단 + 승인 게이트)
```

**3단 핵심 엔진:**
- **Paperclip** = 회사 (조직도, 예산, 결재, 거버넌스)
- **OpenClaw** = 직원 (브라우저/터미널/파일 실무 수행)
- **NemoClaw** = 보안관 (위험 행동 차단, 셀러 승인 대기)

## 7. 신뢰 장벽 & 대응 전략

### 최대 리스크: "AI에 내 사업을 맡겨도 되나?"

| 장벽 | 대응 |
|------|------|
| AI 환각 (잘못된 계산/답변) | 모든 액션에 "승인 게이트" + 실행 로그 투명 공개 |
| 데이터 보안 불신 | 로컬 실행 옵션 + 암호화 + 명확한 통제권 |

| 만능주의 함정 | MVP로 상세페이지 하나만 확실히 → 점진 확장 |

### 페르소나별 신뢰 전략
- **성장기(이성장):** ROI 데이터 + 보안 기술력 논리적 입증

- **퇴직(박퇴직):** 카카오톡 기반 익숙한 인터페이스 + 친절한 가이드

## 9. Go-to-Market 전략

### Phase 1: 검증 (1~2개월)
1. **컨시어지 MVP:** 카톡 채널 개설 → 셀러가 URL 보내면 → 오퍼레이터가 detailpages_2.0으로 결과물 수동 전달 → 지불 의사 검증
2. **시연 영상 3분:** URL→상세페이지 3분 생성 과정 촬영 → 셀러 커뮤니티(셀러노트, 스마트스토어 카페) 배포
3. **사전 예약:** 50명 확보 시 Phase 2 진행

### Phase 2: MVP 런칭 (3~4개월)
1. detailpages_2.0 보안 수정 + 승인 게이트 UI 추가
2. CS 초안 생성기 개발
3. Starter 티어로 오픈 베타 (월 49,000원)
4. 알바봇 마켓 등록 (4/5 강의 활용)

### Phase 3: 확장 (5~12개월)
1. Pro 티어 + 재무/소싱 에이전트 추가
2. Paperclip 조직 템플릿으로 "원클릭 AI 회사 생성"
3. 셀러 성공 사례 → 바이럴 마케팅

## 11. 검증해야 할 핵심 가설

1. **셀러는 복잡한 UI보다 자연어 인터페이스에서 사용 지속성이 높다**
   → 검증: 카톡 컨시어지 vs 웹 대시보드 A/B 테스트

2. **상세페이지 자동 생성 퀄리티가 일정 수준 넘으면 즉각 구독한다**
   → 검증: 무료 5건 제공 후 유료 전환율 측정

3. **AI CS는 "완전자동"보다 "승인 후 발송" 시 도입률 3배 높다**
   → 검증: 두 모드 동시 제공 후 선택 비율 추적

4. **"직원 1명 월급의 1/10" 프레이밍이 "월 구독료" 프레이밍보다 전환율 높다**
   → 검증: 랜딩페이지 A/B 카피 테스트

5. **부업 셀러는 상세페이지가 해결되면 CS→세무 순으로 추가 구매한다**
   → 검증: Starter 사용자의 기능 요청 패턴 추적
