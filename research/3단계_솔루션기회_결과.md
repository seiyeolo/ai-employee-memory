## 1. 니즈 맵핑

|-----------|-------------|-------------|----------|
| 상세페이지 제작 | 외주(10~30만원) 또는 직접(2~3일) | URL만 주면 자동 생성 1분 | 시간 90%↓, 외주비 절감, 전환율↑ |
| 다채널 수기업무 | 사방넷 써도 엑셀 수작업 여전 | 주문→발주→송장 100% 자동화 | 반복노동 제로, 휴먼에러 방지 |

## 2. 기존 솔루션 경쟁 분석

| 솔루션 | 강점 | 약점 | 가격 |
|--------|------|------|------|
| **사방넷** (1위) | 650개+ 채널 연동, 대량 관리 최적화 | 복잡한 UI, 가파른 학습곡선 | 월 ~25만원 (발주 3천건) |
| **플레이오토** | 대용량 등록, 해외 연동 | 별도 직원교육 필요, 소규모 셀러에 무거움 | 월 ~15만원 |
| **이지어드민** | 사용자 친화적 UI, 회계/매출 통계 | 연동 50개로 적음, 복잡 요구 한계 | - |

### 🚨 공백(Gap) 영역
기존 솔루션 = **"분산된 화면을 하나로 모아주는 역할"에 그침**
- 엑셀 변환은 여전히 수작업
- 상세페이지 디자인/번역 → 미지원
- 수익성 분석 → 미지원
- 키워드 최적화 → 미지원
- **매출을 일으키는 '핵심 지능형 업무'는 전혀 지원하지 못함**

## 3. ⭐ "도구 vs 조직" 패러다임 비교

| 구분 | 기존 솔루션 (도구) | AI 에이전트 (조직) |
|------|-------------------|-------------------|
| **작동 방식** | 셀러가 직접 로그인, 클릭, 엑셀 업/다운 | "쿠팡에 등록해줘" 지시만 내림 |
| **학습 비용** | 높음 (수십 장 매뉴얼, 직원 교육) | **제로** (자연어 명령) |
| **시간 절약** | 수작업 N번 → 1번 (선형적) | 사람 개입 단계 소거 (**기하급수적**) |
| **확장성** | 주문 늘면 직원 추가 고용 필요 | AI 자동 스케일링, 추가 채용 불필요 |

> **핵심:** 도구는 "더 빠르게 일하게" 해주지만, 조직은 "일 자체를 대신" 해준다.

| 해외 솔루션 | 핵심 기능 | 한국 현황 |

| **Jungle Scout** | 판매량 예측(~12% 오차), 데이터 기반 소싱 검증 | **없음** (감에 의존) |
| **Shopify 앱** | Growth Suite 원스톱 (소싱→페이지→광고→분석) | **없음** (각각 파편화) |

### 한국 시장의 결핍
> 뒷단(Back-office): 배송/주문 관리 → 사방넷 등으로 발달
> 앞단(Front-office): 소싱 검증 → AI 페이지 생성 → AI 광고 최적화 → **절대적으로 부족**

## 5. 솔루션 기회 TOP 3

### 🏆 1위: AI 자동 상세페이지 & 리스팅 에이전트
- **TAM:** 가장 큼 (모든 셀러의 #1 페인포인트)
- **기술 난이도:** 중간 (OCR, 이미지 번역, 생성형 AI 이미 상용화)
- **경쟁:** 드랩아트 등 있으나 "등록까지 원클릭"은 없음
- **수익 모델:** 건당 과금 또는 월 구독
- **컨셉:** 타오바오 링크 입력 → AI가 한글 번역 + SEO 최적화 + 상세페이지 렌더링 → 셀러는 "승인" 클릭 → 채널 자동 배포

### 🥈 2위: 24/7 다채널 AI CS 매니저
- **TAM:** 감정노동 심각도 최상, 이탈 방지 핵심
- **기술 난이도:** 높음 (각 채널 API + 실제 환불 처리 연동)
- **경쟁:** 단순 챗봇은 많으나 "환불까지 완결 처리"는 없음
- **수익 모델:** 티켓(CS 건수) 단위 과금
- **컨셉:** AI가 고객 분노 문의 필터링 → 배송/환불 규정 기반 자동 답변 → 시스템 내 환불 접수 완결

### 🥉 3위: 한국형 Helium 10 (데이터 소싱 + ROI 예측)
- **TAM:** 높음
- **기술 난이도:** 매우 높음 (네이버/쿠팡 데이터 크롤링)
- **경쟁:** 아이템스카우트 있으나 ROI 시뮬레이션은 없음
- **수익 모델:** 프리미엄 데이터 월 구독 (고단가)
- **컨셉:** 특정 상품 소싱 시 예상 마진, 광고 단가, 월수익 시뮬레이션 → 소싱 불확실성 제거
