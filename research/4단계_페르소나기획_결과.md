## 1. 타겟 셀러 페르소나 (4개)

### ① 김파워 (38세) — 전업 파워셀러
- **카테고리:** 생활용품/소형가전 | **월매출:** 5,000만원
- **플랫폼:** 스마트스토어, 쿠팡, 자사몰 | **운영:** 4년
- **하루 루틴:** 오전 채널별 주문/재고 수집 → 엑셀 통합 → 출고 지시. 오후 광고 효율(ROAS) 분석 → 신제품 소싱
- **3대 고충:** ① 다채널 주문/재고 동기화 수기 한계 ② 직원 채용/관리 인건비+감정소모 ③ 정교한 광고/매출 데이터 분석 어려움
- **바람:** 단순 반복 100% 자동화, 본인은 고마진 상품 기획+브랜딩에 집중
- **현재 툴:** 사방넷+이알피아 복수 사용 (월 25~50만원)

### ② 이성장 (32세) — 성장기 셀러
- **카테고리:** 의류/패션잡화 | **월매출:** 1,000만원
- **플랫폼:** 스마트스토어, 에이블리 | **운영:** 1년 2개월
- **하루 루틴:** 트렌드 서칭 → 촬영 → 상세페이지 디자인 → CS → 택배 포장
- **3대 고충:** ① 상세페이지 1건에 2~3일 (신상품 업로드 병목) ② 배송/환불 CS 감정노동 ③ 혼자 모든 것 처리하는 시간 부족
- **바람:** 디자인과 CS를 믿고 맡길 "가성비 좋은 직원"
- **현재 툴:** 카페24 기본, 이미지 편집, 챗봇 기본형 (월 5~10만원)

### ③ 박퇴직 (58세) — 신규 진입 셀러 (퇴직 후 3개월)
- **카테고리:** 건강기능식품/해외직구 | **월매출:** 150만원
- **플랫폼:** 스마트스토어 | **운영:** 3개월
- **하루 루틴:** 유튜브 구매대행 강의 시청 → 아이템 소싱 시도 → 판매자 센터 UI에서 시간 지체
- **3대 고충:** ① 디지털 문해력 부족 (ICT 활용 어려움) ② 세무/관부가세 복잡성+두려움 ③ 복잡한 솔루션 툴 학습 불가
- **바람:** "이거 팔아줘", "세금 계산해줘" 말로 지시하면 알아서 처리되는 서비스
- **현재 툴:** 엑셀, 무료 유튜브 강의 (월 0~3만원)

### ④ 최부업 (41세) — 부업 셀러 (직장인 투잡)
- **카테고리:** 해외 구매대행 | **월매출:** 300만원
- **플랫폼:** 쿠팡, 11번가 | **운영:** 8개월
- **하루 루틴:** 퇴근 후 밤10시~새벽1시 상품 스크래핑 → 번역/업로드 → 야간 CS
- **3대 고충:** ① 절대적 물리적 시간 부족 (하루 2~3시간) ② 상표권 침해 필터링 어려움 ③ 해외 배송 지연 클레임
- **바람:** 본업 중 쇼핑몰이 알아서 돌아가고, 퇴근 후 최소 승인만 누르기
- **현재 툴:** 픽투셀, 마인 등 반자동 대량등록 (월 10~15만원)

## 2. 핵심 인사이트 요약

### 가장 큰 미충족 니즈 3가지
1. **Front-office 인텔리전스 부재:** 기존 툴은 Back-office(재고/주문)에 치중. 매출 견인하는 소싱/상세페이지/마케팅 최적화는 빈 곳
2. **자연어 지시 환경:** 도구가 복잡해질수록 학습비용↑. "말로 지시 → 결과 확인" 직관적 UX 필요
3. **감정노동/행정 스트레스 해소:** CS 클레임 방어 + 세무/부가세 처리가 에너지를 심각하게 고갈

### 시장 진입 타이밍이 좋은 이유
- 최저가 경쟁 심화 → "인건비 절감"이 생존의 핵심
- 생성형 AI 기술이 실무 적용 가능 수준으로 성숙 (상세페이지 100분→3분)
- 폐업 100만 시대 + 부업 22% 증가 → 생존 도구 수요 급증

### 진입 리스크 TOP 3
1. **AI 환각(Hallucination):** CS 답변이나 세금 계산 오류 시 셀러에게 직접 재무 피해
2. **데이터 보안 불신:** 스토어 계정/매출/고객 정보를 신생 AI에 맡기는 저항감
3. **만능주의 함정:** 초기부터 모든 기능 → 무겁고 완성도 떨어짐. MVP로 하나 확실히 해결

## 3. 신뢰 장벽 분석

### 신뢰 확보 최소 조건
> **AI는 "제안(Draft)"까지만, 최종 실행은 셀러가 "승인" 클릭 = 인간 승인 게이트(Human-in-the-loop) 필수**

- 정보보호에 대한 명확한 통제권 제공
- 투명한 보안 정책 명시
- AI CS 답변: "완전 자동" 수용도 낮음 → **"초안 작성 → 원클릭 승인 → 발송"** 형태가 도입률 3배↑

### 페르소나별 신뢰 구축 차이
| 페르소나 | 신뢰 방식 |
|---------|----------|
| 전업/성장기 | ROI + 데이터 보안 기술력(암호화) 논리적 입증 |
| 신규(퇴직) | 익숙한 인터페이스(카카오톡) + 인간 친화적 가이드 |
| 부업 | 최소 개입(승인만) + 자동화 결과의 일관성 |

## 4. 경쟁 우위 차별화

### 포지셔닝
```
사방넷 = "비싼 도구" (운전법을 배워야 함)
우리   = "가상의 조직" (말만 하면 실무 수행)
셀러   = 실무자 → "팀장(Manager)" 역할만
```

### 비용 설득력
> "최저임금(월 209만원)의 10%도 안 되는 비용으로 24시간 일하는 MD, 디자이너, CS 직원을 고용하세요."
- 인건비 대체 프레임 (SaaS 구독료 프레임 X)
- 퇴사 리스크, 채용 스트레스에 지친 셀러에게 강력한 설득력

### 전환 트리거 (Aha-Moment)
1. 2~3일 걸리던 상세페이지가 **URL 입력 한 번으로 3분** 완성
2. 악성 CS 클레임에 AI가 규정 기반 정중한 답변 초안을 **1초** 제시

## 5. Next Step

### 핵심 가설 3가지
1. 셀러는 복잡한 UI보다 **대화형(자연어) 인터페이스**에서 사용 지속성이 높다
2. 상세페이지 자동 생성 퀄리티가 일정 수준 넘으면, 성장기/부업 셀러가 **즉각 월 구독** 한다
3. AI CS 응대는 완전 자동보다 **"승인 후 발송"** 기능 시 도입률 **3배** 높다

### 검증 방법
- **영상 MVP:** 3분짜리 시연 영상 → 셀러 커뮤니티 배포 → 사전 예약자 확보
- **컨시어지 MVP:** 카톡으로 지시 → 오퍼레이터가 AI로 수작업 결과물 제공 → 지불 의사 검증

### MVP 필수 기능 3가지
1. **AI 상세페이지 에이전트:** 소싱 URL → 이미지 크롭/번역/배치 + 마케팅 문구 자동 생성
2. **CS 답변 초안 생성기:** 고객 문의 수집 → 최적 답변 대기 → 원클릭 승인 발송
3. **자연어 데이터 랩:** "오늘 쿠팡 마진 얼마야?" → 텍스트/표로 즉답 (대시보드 대체)

### 가격 모델 초안
| Tier | 대상 | 월 가격 | 포함 내용 |
|------|------|--------|----------|
| **Starter** | 부업/신규 | **49,000원** | 상세페이지 N건, 기본 CS 초안 |
| **Pro** | 전업/성장기 | **199,000원** | 무제한 AI 에이전트, 광고/마진 분석, 다국어 번역 |

> **Pro 기준 = 직원 1명 월급(209만원)의 1/10**
