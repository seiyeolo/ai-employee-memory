## 🔧 도구 사용 규칙 (필수)

### 사용 가능한 도구
| 도구 | 용도 |
|------|------|
| `read` | 파일 읽기 |
| `write` | 파일 생성/덮어쓰기 |
| `edit` | 파일 부분 수정 |
| `exec` | 셸 명령 실행 |

### ⚠️ 핵심 규칙
1. **"했습니다"라고만 답하지 마라** — 반드시 도구를 호출하여 실제 파일을 수정하라
2. **수정 후 `read`로 검증하라** — 변경이 실제로 반영됐는지 확인
3. **CSV 형식 엄수** — 헤더 유지, 쉼표 구분, 기존 행 보존

### 업무별 파일 수정 방법

#### 출고/발송 기록 시
1. `read("data/출고이력.csv")` → 현재 기록 확인
2. `edit("data/출고이력.csv")` → 새 행 추가: `날짜,유형,코드,품명,수량,거래처,비고`
3. `read("data/출고이력.csv")` → 변경 확인

#### 납품 기록 시
1. `edit("data/출고이력.csv")` → 유형을 "납품"으로 새 행 추가
2. 해당 납품체크리스트 파일 업데이트

### 데이터 파일 형식

#### data/출고이력.csv
```csv
날짜,유형,코드,품명,수량,거래처,비고
2026-03-30,택배,PG-001,퍼티스트 골프 고급형,3,테스트골프장,
```
