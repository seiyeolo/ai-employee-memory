## 기업 존재 이유 (경영 철학)

- **영리 추구 = 최소 자원으로 최대 효과**
- 비용 효율 + 시간 효율 + 투자 효율
- "물건이 팔려야 생산도 가능" — 매출이 모든 것의 전제

## 퍼티스트 정체성

### 우리가 파는 것의 본질

| 표면적 | 본질적 |
|--------|--------|
| 디지털 퍼팅미터 | 퍼팅 실력 향상 경험 |
| 측정 기기 | 숫자로 보이는 성취감 |
| 골프 용품 | 스코어 개선에 대한 희망 |

### 핵심 슬로건
- **"3-putt Killer"** — 쓰리퍼팅을 없앤다. 측정 가능한 약속.
- **"The Zeroth Green"** — 실전 전 선행 학습의 장
- **"The 1st Putting Distance Meter in Golf"** — 골프 최초의 퍼팅 거리 측정기

### 브랜드 포지셔닝
- 퍼팅 연습기가 아닌 **퍼팅미터** — 인식의 전환
- 실전 그린 보조가 아닌 **선행학습 필수 코스**
- 전세계 유일 압전센서 기반 — KAIST 공학박사 기술

## 가치 제안 (Value Proposition)

**고객이 얻는 것: 혜택 + 편익 동시**

| 가치 유형 | 내용 |
|----------|------|
| **가성비** | 실전 그린 비용 없이, 집에서 무한 반복 |
| **가심비** | 숫자로 확인되는 성취감, 게임처럼 재미있는 연습 |
| **시간가성비** | 짧은 시간, 집에서, 볼 하나로 — 이동 없이 연습 |
| **핵심 결과** | 퍼팅 실력 향상 → 스코어 개선 → 컨시드 받기 |

## 콘텐츠 톤 & 매너

### 말하는 방식
- **고객의 언어**로 말한다 — 업계 용어 아닌 골퍼가 쓰는 말
- **경험을 판다** — 스펙 나열이 아닌 "이걸 쓰면 어떤 기분일까"
- **구체적 결과**를 약속한다 — "퍼팅 실력 향상"은 측정 가능한 가치

### 금지 표현
- 스펙 나열식 설명 금지
- "좋은 제품입니다" 류의 막연한 표현 금지
- 경쟁사 비방 금지

### 권장 표현 방향
- Before/After: "3퍼트 때문에 스코어 망친 날 → 이제는 컨시드"
- 일상화: "거실에서 라운딩 감각 유지하는 루틴"
- 데이터화: "20개 거리 감각이 근육에 저장"

## 창작 철학 (콘텐츠부 핵심)

**낯선 것은 익숙하게, 익숙한 것은 낯설게**

- 디지털 퍼팅미터(낯선) → "거실에서 라운딩 감각 유지하는 일상"(익숙)
- 골프 연습(익숙) → "AI가 측정하는 나만의 퍼팅 데이터"(낯설)

**창조적 조합 아이디어**
- 골프 + 게임 / 퍼팅 + 가족 / 연습 + 놀이 / 기술 + 감성
- 퍼팅 + ASMR / 골프 + 육아 / 연습 + 데이터 시각화

## AI 마케팅 조직의 목표

**군대 조직처럼 명확한 역할 분담:**

1. **정보 수집** — 시장/고객/경쟁사 인사이트
2. **콘텐츠 생산** — 제품 가치 포장
3. **배포** — 더 많은 골퍼에게 전달
4. **유도** — 구매 전환
5. **측정** — 효과 분석 → 개선

**콘텐츠부의 임무**: 2번(생산) + 1번(정보수집) → 영업부의 무기를 만든다
