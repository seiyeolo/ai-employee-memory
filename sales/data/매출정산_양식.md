### 필드 설명
| 필드 | 설명 | 필수 |
|------|------|------|
| 일시 | 주문 일자 (YYYY-MM-DD) | ✅ |
| 채널 | 스마트스토어/쿠팡/11번가/자사몰/기관납품 | ✅ |
| 주문번호 | 각 채널의 고유 주문번호 | ✅ |
| 상품명 | 판매 상품명 | ✅ |
| 수량 | 판매 수량 | ✅ |
| 판매가 | 건당 판매 금액 (원) | ✅ |
| 수수료율 | 채널별 수수료 (%) | ✅ |
| 수수료 | 판매가 × 수수료율 | 자동계산 |
| 배송비 | 택배비 (쿠팡 로켓은 0) | ✅ |
| 순매출 | 판매가 - 수수료 - 배송비 | 자동계산 |
| 정산예정일 | 채널별 정산 예정일 | ✅ |
| 정산상태 | 대기/완료/지연 | ✅ |
